debian10升级debian11

更新现有软件包

apt update && apt upgrade -y

删除未使用的依赖项：

apt --purge autoremove

检查安装的包的一致性（正常情况下无输出）

dpkg -C

查看无法升级（on hold）的包（正常情况下无输出）

apt-mark showhold

更新源中的系统代码

sudo cp /etc/apt/sources.list /etc/apt/sources.list.bak && sudo sed -i 's/buster/bullseye/g' /etc/apt/sources.list

开始升级（遇到覆盖配置的提示时，按D看看差异）

sudo apt update && sudo apt full-upgrade

查看更新是否成功

cat /etc/os-release

清理旧包

1

sudo apt --purge autoremove && sudo apt autoclean

移除旧内核

查看有哪些内核

1

dpkg -l | tail -n +6| grep -E 'linux-image-[0-9]+'| grep -Fv $(uname -r)

输出的内容中包括了 Kernel 的三种状态：

  • rc：表示已经被移除
  • ii：表示符合移除条件（可移除）
  • iU：已进入 apt 安装队列，但还未被安装（不可移除）。

选择需要删除的内核

1

sudo dpkg --purge linux-image-xxxx
